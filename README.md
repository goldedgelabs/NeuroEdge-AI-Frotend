# NeuroEdge — Complete Final Frontend

This bundle is a comprehensive frontend scaffold for NeuroEdge (mobile-first, PWA-ready).

Run locally:

```bash
npm install
npm run dev
```

Notes:
- Uses mock API by default; flip USE_MOCK in src/lib/api.js or set VITE_API_BASE to connect a backend.
- Service worker included. In dev, reload to see SW changes.
- Assets upload stored in IndexedDB until backend connected.
