# NeuroEdge AI — Frontend

NeuroEdge AI is a fully-featured PWA-ready AI interface with:
- Multi-agent chat system  
- Tools (Web Search, File Upload, Vision, Code)  
- Sidebar + TopBar layout  
- Settings Drawer (model, temperature, agent config)  
- ReasoningGraph visualizer  
- Firebase integration  
- Offline mode (service worker + PWA manifest)

---

## 🚀 Tech Stack

- **React 18 + Vite**
- **TailwindCSS**
- **Firebase**
- **PWA Support**
- **ShadCN/UI**
- **Lucide Icons**
- **Framer Motion**
- **Recharts**
